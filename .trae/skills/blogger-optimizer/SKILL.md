## 快速执行

```bash
node d:/opencli/douyin-videos/scripts/utils/blogger-optimizer.js
```

**输出报告**：`reports/blogger-optimization-{YYYY-MM-DD}.md`

---

## 触发条件

| 场景 | 触发 |
|------|------|
| 博主添加时 | ✅ |
| Stage2 采集完成后 | ✅ |
| Content-Cleaner 完成后 | ✅ |
| 用户提到"优化博主"、"初筛"、"评估"、"黑名单"等关键词 | ✅ |

---

## 核心判断标准

| 条件 | 处理 |
|------|------|
| 有高质量视频（quality="accessible"） | 休眠保留，永不删除 |
| 无高质量视频 + 低评分（<30） | 加入黑名单 |
| 过滤比例高（视频太短被淘汰） | 扣分，降低评估 |

---

## 评分维度（参考）

报告提供九维评分作为辅助参考，**最终决策由 AI 综合判断**：

| 维度 | 权重 | 说明 |
|------|------|------|
| 内容相关度 | 30% | 基于 relevance_score |
| 质量得分 | 15% | 高质量比例 × 样本量惩罚 |
| 采集效率 | 10% | 实际采集情况 |
| 内容一致性 | 10% | 主题稳定性 |
| 过滤结构 | 5% | 过滤原因分布 |
| 粉丝价值 | 5% | 粉丝数对数值 |
| 平台视频因子 | 5% | 视频数量 |
| 广告扣分 | - | 广告内容扣分 |
| 过滤扣分 | - | 视频太短被淘汰 |

---

## 数据文件（参考）

| 文件 | 用途 |
|------|------|
| `data/bloggers.json` | 博主列表 |
| `data/filter-report.json` | 过滤报告（视频太短被淘汰统计） |
| `data/blogger-blacklist.json` | 黑名单 |
| `results/{博主}/metadata.json` | 博主详情 |
